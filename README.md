# 作曲から逃げるな

作曲テーマに迷ったときのための、シンプルな抽選ツールです。

リファレンスアーティスト、BPM、曲のセクションをランダムに表示します。ボタンを押すたびに新しいお題を抽選できるので、「今日は何を作るか」をすばやく決められます。

Python サーバーなしで動く静的 Web アプリなので、`index.html` をブラウザで開くだけで使えます。

## 機能

- リファレンスアーティストを重み付きランダムで抽選
- BPM を 80 から 200 の範囲で抽選
- 「イントロ」「Aメロ」「サビ」から制作セクションを抽選
- ブラウザ上で再抽選
- ブラウザ上でリファレンス、重み、BPM 範囲、セクション候補を編集
- 設定をブラウザに自動保存
- サーバー不要でブラウザだけで動作

## 必要なもの

- Web ブラウザ

## 使い方

`index.html` をブラウザで開いてください。

Windows では `作曲から逃げるな.bat` をダブルクリックして開くこともできます。

## 主な構成

```text
.
├── index.html
├── static/
│   ├── app.js
│   └── style.css
└── 作曲から逃げるな.bat
```

## 抽選内容の変更

ページ内の「抽選設定」から変更できます。

- リファレンス: 名前と重みを編集できます
- 重み: 数値が大きいほど抽選されやすくなります
- BPM: 最小値と最大値を編集できます
- セクション: 改行区切りで候補を編集できます

設定はブラウザに自動保存されます。初期状態に戻したい場合は「初期設定に戻す」を押してください。
